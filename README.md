# 🤖 Chatbot Web Application

## 📌 Overview

This project is a simple web-based chatbot that interacts with users and responds to predefined queries.

## 🎯 Objective

To build an interactive chatbot interface and understand basic frontend development.

## 🚀 Features

* Interactive chat interface
* Instant responses to user queries
* Predefined response system
* Clean and simple UI

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

## 🧠 Working Principle

1. User types a message
2. JavaScript captures input
3. Input is matched with predefined responses
4. Bot replies accordingly

## ▶️ How to Run

1. Open project folder
2. Double-click index.html
   OR
3. Use Live Server in VS Code

## 📊 Example

User: Hi
Bot: Hello!

User: How are you
Bot: I am fine!
